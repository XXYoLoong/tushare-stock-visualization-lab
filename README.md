# Tushare 股票数据可视化分析实验

基于 Tushare 的股票日线行情分析、收益率对比、趋势识别与交易信号可视化实验项目。

## 项目内容

- 题目一：单只股票 2025 年日线行情清洗、统计与可视化
- 题目二：三只股票 2025 年累计收益率对比与均线分析
- 题目三：最近 1 年趋势识别、均线信号与简单交易规则分析
- 自动生成图表、结果数据与 Word 实验报告

说明：题目四在原作业中标注为选做项，本项目默认完成前三道必做题并生成提交所需报告。

## 目录结构

```text
docs/                  过程文档与说明
src/                   源代码
outputs/
  charts/              导出的图表
  data/                生成的数据结果
  report/              Word 报告
```

## 环境准备

1. 安装 Python 3.11+。
2. 安装依赖：

```bash
pip install -r requirements.txt
```

3. 复制 `.env.example` 为 `.env`，填写 `TUSHARE_TOKEN`。

## 运行

```bash
python -m src.main
```

运行后将生成：

- 图表文件
- 清洗后的数据结果
- Word 实验报告

## 说明

- 默认股票：
  - 题目一：贵州茅台 `600519.SH`
  - 题目二：中国平安 `601318.SH`、比亚迪 `002594.SZ`、贵州茅台 `600519.SH`
  - 题目三：比亚迪 `002594.SZ`
- Tushare token 不会提交到 Git 仓库。
